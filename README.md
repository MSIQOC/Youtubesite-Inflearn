# 유튜브 사이트 만들기
백엔드를 공부하면서 프론트엔드의 동작 과정도 함께 배워야하는 과정이 필요하다고 생각해서 인프런의 유튜브 사이트 만들기 강의를 참고해서 공부해보았습니다.
